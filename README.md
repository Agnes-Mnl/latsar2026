Tahapan Penggunaan Mini Sistem

A. Bagi Sub Bagian Pengelolaan Data (PUSAT - Biro SDM)

1. Download Zip folder Mini Sistem di Github https://github.com/Agnes-Mnl/latsar2026
<img width="1895" height="877" alt="image" src="https://github.com/user-attachments/assets/67ce1804-d5e5-422c-8d88-0317f34a6083" />

2. Download aplikasi Visual Studio Code melalui https://code.visualstudio.com/download
3. Install python di PC/Komputer yang digunakan melalui https://www.python.org/downloads/ 
4. Lakukan instalasi mini sistem di visual studio code dengan cara masuk ke terminal di visual studio code dan ketik berikut :
   a. cd code

   b. python -m venv .venv

   c. .venv\Scripts\activate

   d. pip install -r requirements.txt

   e. python app.py
6. Lalu klik Link yang dihasilkan : http://127.0.0.1:5000 . Link ini dapat diakses secara lokal saja
