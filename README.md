Tahapan Penggunaan Mini Sistem

A. Mini Sistem
I. Instalasi Mini Sistem di PC/Komputer

1. Download Zip folder Mini Sistem di Github https://github.com/Agnes-Mnl/latsar2026
<img width="1895" height="877" alt="image" src="https://github.com/user-attachments/assets/67ce1804-d5e5-422c-8d88-0317f34a6083" />

2. Download aplikasi Visual Studio Code melalui https://code.visualstudio.com/download
3. Install python di PC/Komputer yang digunakan melalui https://www.python.org/downloads/ 
4. Lakukan instalasi mini sistem di visual studio code dengan cara masuk ke terminal di visual studio code dan ketik berikut:
   
   a. cd code

   b. python -m venv .venv

   c. .venv\Scripts\activate

   d. pip install -r requirements.txt

   e. python app.py
6. Lalu klik Link yang dihasilkan : http://127.0.0.1:5000 . Link ini dapat diakses secara lokal saja. Berikut tampilan mini sistem yang dapat digunakan
   <img width="1908" height="918" alt="image" src="https://github.com/user-attachments/assets/f59720e7-bf99-471b-b93d-1b24935f64c0" />

II. Penggunaan menggunakan wifi yang sama

1. 1 Operator akan melakukan instalasi di PC/Komputer pribadi dengan menjalankan mini sistem di Localhost http://127.0.0.1:5000
2. Pengguna lain yang ingin menggunakan mini sistem dapat menggunakan wifi yang sama dengan operator dan mengakses link http://127.0.0.1:5000
3. Pengguna dapat menggunakan mini sistem
